# Computer Science

Computing fundamentals and systems theory.

## Overview

Computer science encompasses algorithms, systems, theory, and practical computing.

## Subjects

### Algorithms
Sorting, searching, graph algorithms, dynamic programming, complexity analysis.

### Operating Systems
Process management, memory management, file systems, concurrency.

### Distributed Systems
Consensus, clocks, fault tolerance, replication, Byzantine systems.

### Databases
Relational theory, indexing, transactions, query optimization.

### Networks
Protocols, routing, transport layer, application layer.

### Security
Cryptography, authentication, access control, system hardening.

### Compilers
Lexical analysis, parsing, semantic analysis, code generation.

## Learning Foundations

- Start with algorithms and complexity
- Move to systems and architectures
- Explore specialization areas (networks, databases, etc.)
