# Computer Science

Computing fundamentals and systems theory.

## Subjects

- **Algorithms** — Sorting, searching, graphs, dynamic programming
- **Operating Systems** — Processes, memory, concurrency
- **Distributed Systems** — Consensus, replication, fault tolerance
- **Databases** — Relational theory, indexing, transactions
- **Networks** — Protocols, routing, transport
- **Security** — Cryptography, authentication, hardening
- **Compilers** — Parsing, semantic analysis, code generation

## Key Principles

- Think in terms of complexity and efficiency
- Consider scale and practical constraints
- Design for reliability and maintainability
