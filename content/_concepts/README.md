# Cross-Domain Concepts

Fundamental concepts that apply across multiple domains.

These concepts provide bridges between different domains and enable cross-domain thinking.

## Concepts

- **[[information-theory]]** — Entropy, coding, communication
- **[[optimization]]** — Convex optimization, gradient descent, Lagrange multipliers
- **[[time-series]]** — Temporal patterns, forecasting, dynamics
- **[[graph-theory]]** — Graphs, networks, algorithms
- **[[game-theory]]** — Strategic interaction, equilibrium, mechanisms
- **[[statistical-inference]]** — Estimation, hypothesis testing, Bayesian methods

## Usage

Reference cross-domain concepts using the `_concepts` folder:

```markdown
See [[optimization]] for general principles, then apply to [[neural-network-training]].
```
