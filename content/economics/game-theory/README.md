# Chapter: Game Theory (Strategic Interaction)

Welcome to the **Game Theory** module. This chapter mirrors the 58-lecture NPTEL syllabus of **"Strategy: An Introduction to Game Theory"** by Aditya Jagannatham.

Game theory is the mathematical framework designed to analyze situations of **strategic interdependence**. In these environments, the optimal course of action for any individual agent depends directly on the choices made by all other agents. This differs fundamentally from standard single-agent optimization (such as maximizing utility over a fixed budget) because the "environment" itself consists of rational, active decision-makers.

## 🎯 Learning Objectives

By the end of this module, you will be able to:
1.  **Formulate strategic situations** as formal mathematical games in both normal (strategic) and extensive (sequential) forms.
2.  **Apply solution concepts** (IESDS, Pure/Mixed Nash Equilibrium, SPNE, BNE, PBE, and ESS) to solve for optimal behaviors.
3.  **Analyze economic applications** including oligopoly duopolies (Cournot, Stackelberg), auction structures (first/second price, all-pay), and bargaining protocols.
4.  **Incorporate information structures** (perfect vs. imperfect, complete vs. incomplete) into strategic plans, evaluating signaling mechanisms and credible threats.
