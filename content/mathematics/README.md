# Mathematics

Foundation of quantitative thinking and abstract reasoning.

## Subjects

- **Probability** — Random variables, distributions, limit theorems
- **Linear Algebra** — Vectors, matrices, eigenvalues
- **Optimization** — Convex optimization, gradient methods
- **Real Analysis** — Limits, continuity, derivatives
- **Statistics** — Estimation, inference, testing

## Learning Path

Start with probability fundamentals, move to linear algebra, then explore applications in optimization and statistics.

## Key Connections

- **To AI:** Linear algebra and probability are essential
- **To Economics:** Statistics and optimization for modeling
- **To Physics:** Calculus and differential equations
